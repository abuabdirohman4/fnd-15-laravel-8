## About Project

Blog that created with admin dashboard & guest view page, there's some features :
- CRUD Posts & Categories
- Upload Image
- Authentication Login & Register
- Authorization Admin & Guest
- Searching Posts & Categories
- Pagination
- Slug Generator
- Trix Editor
- Clockwork & N+1 Problem
